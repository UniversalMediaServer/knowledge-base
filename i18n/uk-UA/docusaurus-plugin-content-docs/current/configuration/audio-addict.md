# Радіомережа Audio Addict Network

**Audio Addict Network** це колекція преміум радіостанцій Інтернету, зокрема:

Класичне радіо, загальножанрове, Рок радіо, джазове радіо, Радіо для релаксації та DI.fm.

Платний **преміум-акаунт** необхідний для доступу до цих радіостанцій. Вам потрібно зареєструвати лише на одному із сайтів, це ім'я користувача та пароль надасть доступ до них всіх.

UMS автоматично зчитує всі доступні сайти radio AudioAddict, їхні канали та фільтри. **Обрані** канали також підтримуються.

## Налаштування

Щоб увімкнути підтримку AudioAddict , додайте наступні рядки до вашого файлу `UMS.conf`:

```
# TRUE = Надати перевагу європейському стрімінговому серверу.
audio_addict_europe = TRUE or FALSE
```

## Розташування

Тека Audio Addict Radio Network розташовано у Вашій бібліотеці медіафайлів Звідси ви можете переглядати та отримувати доступ до всіх доступних станцій напряму.
